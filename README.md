# EE-FE: First Year Electrical Engineering Syllabus

## Overview
This repository contains the proposed syllabus for **First Year B.E. Electrical Engineering** program, designed for implementation from **Academic Year 2026-27**.

## Key Features
- **NEP-2020 Compliant** - Follows AICTE guidelines and credit structure
- **Industry Relevant** - Updated content for 2026 and beyond
- **Balanced Approach** - Traditional core + Modern technologies

## Course Structure

### Semester I (20 Credits)
| Course | Credits | Type |
|--------|---------|------|
| Engineering Mathematics I | 4 | BSC |
| Engineering Physics + Lab | 3+1 | BSC |
| Fundamentals of Electrical Engineering + Lab | 3+1 | ESC |
| Programming with Python | 3 | ESC |
| Electrical Workshop & Safety | 3 | VSEC |
| Professional Communication | 2 | AEC |

### Semester II (20 Credits)
| Course | Credits | Type |
|--------|---------|------|
| Engineering Mathematics II | 4 | BSC |
| Engineering Chemistry + Lab | 3+1 | BSC |
| Basic Electronics Engineering + Lab | 3+1 | PCC |
| Renewable Energy Fundamentals | 3 | VSEC |
| Engineering Graphics & CAD | 3 | ESC |
| Environmental Studies | 2 | MDC |

## Highlights for 2026
- **Python Programming** - Industry standard for engineering computations
- **Renewable Energy** - Dedicated course covering Solar PV, Wind, Energy Storage
- **Modern Chemistry** - Li-ion batteries, Fuel cells, Green hydrogen
- **Electrical Safety** - Practical skills with IE Rules compliance
- **Simulation Tools** - LTSpice, MATLAB integration in labs
